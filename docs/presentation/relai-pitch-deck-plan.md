# RelAi 발표자료 컨셉 정리

## 목적
- 공모전/데모 발표에서 `RelAi`를 "AI 챗봇"이 아니라 `요구사항 변경을 팀의 실행 단위로 정렬하는 협업 운영 시스템`으로 인식시키는 것
- 문제 제기 -> 근거 -> 솔루션 -> 제품 흐름 -> 차별성 -> 사업성 -> 비전 순으로 설득 구조를 고정하는 것
- 내일 바로 PPT 제작에 들어갈 수 있도록 슬라이드별 메시지와 자산 사용 위치를 정리해두는 것

## 톤앤매너
- 흰 배경, 검정 텍스트 중심
- 회색은 최소한으로만 사용
- 포인트 컬러는 아주 약한 블루만 사용
- "AI 채팅앱"처럼 보이지 않게 하고, 운영 보드/의사결정 시스템처럼 보이게 구성
- 한 슬라이드에 한 메시지만 남기고 문장은 짧고 강하게

## 핵심 한 줄
`RelAi는 요구사항 변경과 팀 논의를 AI가 실행 가능한 합의 구조로 정렬하는 협업 운영 시스템이다.`

## 발표 흐름

### 1. 표지
- 제목: `RelAi`
- 부제: `요구사항 변경을 팀의 실행 단위로 정렬하는 AI 협업 운영 시스템`
- 하단 키워드: `alignment / memory / impact / approval / orchestration`

### 2. 문제 정의
- 현업에서 실제로 반복되는 문제를 바로 던지기
- 메시지:
  - 기획이 바뀌면 개발이 흔들린다
  - 품질 검수가 들어오면 다시 수정이 열린다
  - 문제는 역량 부족보다 합의 구조 부재에 있다
- 넣을 요소:
  - 기획자 / 개발자 / QA / 승인권자 흐름 다이어그램
  - 재작업, 승인 지연, 회의 반복 같은 단어 강조

### 3. 왜 지금인가
- 개인의 AI 활용은 빨라졌는데 팀의 정렬 방식은 그대로라는 메시지
- 활용 문장:
  - AI는 각 개인의 속도를 높였지만 팀의 합의 구조를 자동으로 개선하지는 못했다
- 넣을 요소:
  - AI 활용 증가 수치
  - 의사소통 오류/재작업 수치
  - "개인 생산성 상승 vs 조직 정렬 정체" 대비

### 4. 인사이트
- 핵심 메시지:
  - 진짜 AX는 개인이 AI를 잘 쓰는 것이 아니라 조직의 의사소통이 AI 흐름에 맞게 빨라지는 것이다
- 여기서 RelAi의 존재 이유를 짧게 연결

### 5. 솔루션 소개
- RelAi 정의를 한 장으로 정리
- 입력:
  - 요구사항 문서
  - 회의 기록
  - 채팅
  - 변경 요청
- 출력:
  - 결정
  - 질문
  - 승인
  - 영향
  - 액션
  - 메모리

### 6. 핵심 플로우
- 발표 핵심 장표
- 흐름:
  - 초안 등록
  - 역할별 해석 변환
  - 합의 잠금 보드
  - 변경 영향 맵
  - 프로젝트 메모리
  - AI 회의 개설
- 여기서는 텍스트보다 박스와 화살표 중심으로 구성

### 7. 제품 화면
- 실제 MVP 화면을 보여주는 장
- 우선순위 화면:
  - 대시보드
  - 변경 영향 맵
  - 프로젝트 메모리
  - AI 회의 개설
- 한 화면 전체를 크게 넣기보다 2x2 조합이 더 보기 좋음

### 8. 차별성
- Mantis / Confluence / 문서 / 메신저와 비교
- 메시지:
  - 기존 도구는 기록을 남긴다
  - RelAi는 그 기록을 다시 해석해 다음 실행을 정렬한다
- 표 비교 예시:
  - 남기는 것 vs 정렬하는 것
  - 저장 vs 판단 지원
  - 문서화 vs 실행 단위 변환

### 9. 사업화
- 초기 고객:
  - 제품팀
  - 개발팀
  - PM 조직
  - SI/외주 협업팀
- 수익 모델:
  - 팀 단위 구독
  - 프로젝트 단위 과금
  - 엔터프라이즈 플랜
  - 부서 확장 패키지

### 10. 실현 가능성 / 검증
- 현재 MVP 개발 상태
- 실제 업무 흐름 기반 검증 중이라는 점
- 추후 연동 확장:
  - Jira
  - Notion
  - Slack
  - Teams
  - GitHub
  - Mantis
  - Confluence

### 11. 기대 효과
- 재작업 감소
- 변경 검토 시간 단축
- 승인 지연 감소
- 회의 후 결정 누락 감소
- 프로젝트 메모리 축적

### 12. 클로징
- 마지막 문장 추천:
  - `RelAi는 AI 시대의 더 빠른 변경을, 더 빠른 합의로 바꾸는 시스템입니다.`
- 또는
  - `개인의 AI 활용을 넘어, 조직의 의사소통 속도까지 끌어올리는 것이 RelAi의 목표입니다.`
